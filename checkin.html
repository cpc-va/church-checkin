    About CPC Check-In App

    Version 0.1

    This app is designed to make CPC attendee check-in simple and fast.

    • Upload your attendees.csv file to get started.
    • Search attendees by name or Chinese name using the search bar.
    • Check in attendees by tapping their name in the list.
    • Add new attendees using the "Add New Attendee" button.
    • Export updated attendees CSV and checked-in attendees CSV as needed.
    • The app can remember checked-in attendees in the browser using local storage.
